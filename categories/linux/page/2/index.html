Mint 19 і сканер Epson Perfection V33

На замовчування програма iscan в Linux Mint не працює зі сканером Epson Perfection V33. Щоб вирішити цю проблему, винонуємо наступні дії: sudo ln -sfr /usr/lib/sane/libsane-epkowa* /usr/lib/x86_64-linux-gnu/sane та сворюємо правило для підключення сканера /etc/udev/rules.d/79-udev-epson.rules: # chmod device EPSON group ATTRS{manufacturer}=="EPSON", DRIVERS=="usb", SUBSYSTEMS=="usb", ATTRS{idVendor}=="04b8", ATTRS{idProduct}=="*", MODE="0777" Посилання https://forums.linuxmint.com/viewtopic.php?t=275853
