Migrate to Hugo from Jekyll

Move static content to static Jekyll has a rule that any directory not starting with _ will be copied as-is to the _site output. Hugo keeps all static content under static. You should therefore move it all there. With Jekyll, something that looked like ▾ <root>/ ▾ images/ logo.png should become ▾ <root>/ ▾ static/ ▾ images/ logo.png Additionally, you’ll want any files that should reside at the root (such as CNAME) to be moved to static. ...

Встановлення Telegram на Debian 9

Для встановлення Telegram на Debian 9 можна скористатись Ubuntu PPA. Для цього виконаємо наступну послідовність: Додаємо репозиторій в /etc/apt/source.list.d/telegram.list deb http://ppa.launchpad.net/atareao/telegram/ubuntu xenial main deb-src http://ppa.launchpad.net/atareao/telegram/ubuntu xenial main Додаємо в систему відкритий ключ PGP до цього PPA apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 36FD5529 За необхідності встановлюємо dirmngr apt install dirmngr Оновлюємо список пакетів apt update Встановлюємо Telegram apt install telegram Посилання https://www.linuxbabe.com/debian/3-ways-install-telegram-messenger-debian-9-stretch
