Заміна даних в MariaDB

Швидкий спосіб виправити текстові дані в БД: Прибрати перехід на новий рядок (\n) UPDATE catalog_product_entity_varchar SET value = REPLACE(value, "\n", "") WHERE attribute_id = 73; Прибрати зайві пробіли UPDATE catalog_product_entity_varchar SET value = REGEXP_REPLACE(value, '[[:space:]]+', ' ') WHERE attribute_id = 73; Увага!!! [[:space:]] працює тільки в MariaDB
